# compose-api-tutorial
Simple Android Compose API Example

The code requires json-server with a data.json file containing the following:

json-server --host [Your Network IP] data.json

```json
{
  "users": [
    {   
      "id": 1,
      "profile": {
        "age": 20, 
        "name": "Ethan",
        "email": "example@email.com"
      }   
    }   
  ]
}
```
